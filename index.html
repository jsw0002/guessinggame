Guessing Game

This is a guessing game where the user inputs his/her name and guesses a number between 1-10.

If the user does not guess correctly the game will give them a hint whether the number is higher or lower than their previous guess.

When the user finally gets the correct answer they are then informed of how many guesses it took them as well as how many guesses it took a computer player to get it right as well.
